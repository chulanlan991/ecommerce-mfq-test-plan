# ecommerce-mfq-test-plan
电商系统 MFQ 测试方案仓库

---

## 项目简介
本仓库存放电商系统的 MFQ 测试计划、测试用例与相关文档，聚焦订单与库存核心业务流程，为功能测试、回归测试提供完整依据。

##  仓库文件说明
| 文件名 | 说明 |
|--------|------|
| `订单与库存系统测试用例.xmind` | 订单创建/支付/取消、库存扣减/回补等场景的思维导图式测试用例 |

##  核心测试范围
- **订单模块**：创建订单、支付流程、订单取消、超时关闭、状态流转
- **库存模块**：下单扣减、支付回补、并发超卖控制、库存预警
- **边界场景**：库存为0、金额为0、重复提交订单、并发下单等异常用例

## 📌 使用说明
1.  `xmind` 文件可直接导入 XMind 查看和编辑
